## 概要
規格外の農作物を生産者と消費者が直接取引できるサービスです。  
アプリ名はmerico(メリコ)です。  
※デモアプリのため実際に商品を購入することはできません。
## 開発環境
* php
* jquery
* scss
* Mysql
